Instalation:

Copy file in root site folder. 
Create one folder "/imgblog/" and set chmod 666 or 777.
Create one folder "/maranlog/" and set chmod 666 or 777.
Set file myblog.db with chmod 666.
Set file comments.db with chmod 666.
Set file /shoutbox/data.dat with chmod 666.
